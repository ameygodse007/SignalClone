## Install & Build

First, make sure you have Expo CLI installed: `npm install -g expo-cli`

Install: `yarn` or `yarn install`

Run Project Locally: `yarn dev` or `yarn start`

## Features

- Expo SDK 41
- iOS, Android and PWA (Progressive Web App)
- React Navigation v4
- PropTypes
